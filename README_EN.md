## ✨ Core Features

| 🧠 Intelligent Interaction | 🎨 Visual Style | 📱 Cross‑Platform |
|----------------------------|-----------------|-------------------|
| **Eye tracking** – gaze highlights icons with animation | **Glassmorphism UI** – real camera passthrough background | **Responsive layout** – perfect on phone/tablet/desktop |
| **Hand pinch** – air‑tap to open apps | **Multiple wallpapers** – beach/mountain/American bedroom | **Gyroscope integration** – tilt device for 3D icon rotation |
| **Seamless tap‑to‑start** – no button needed, natural interaction | **Round thumbnail panel** – click mountain icon for elegant switching | **Free deployment** – one‑click publish to Netlify |

## 🛠️ Running Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/OrangeChen011/VisionOSSimu-html.git
2。**Enter the directory**
   cd VisionOSSimu-html
3.**Start a local server (camera API requires localhost or HTTPS)**
   # Using Python 3
python -m http.server 8000
# Or using Node.js
npx serve .
4.**Open in browser：http://localhost:8000**
